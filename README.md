# 🍧 Dapur Jejenz — Order Web App

Aplikasi order menu Dapur Jejenz via WhatsApp.

## ✨ Fitur

- 🛒 Order menu dengan varian (Original, Jumbo, dll)
- 💰 Diskon otomatis 10% untuk order ≥ Rp 50.000
- 📱 Checkout WhatsApp otomatis
- 📊 Dashboard admin (terkunci PIN)
- 🖼️ Upload foto per menu
- 📤 Export JSON / CSV / PDF
- 🌙 Mode gelap/terang
- 💾 Data persistent di localStorage

## 🚀 Cara Pakai

1. Buka `index.html` di browser
2. Pilih menu, varian, klik **Tambah** ke keranjang
3. Isi nama, checkout via WhatsApp

## 🔐 Admin

1. Klik tab **🔒 Admin**
2. Masukkan PIN: `1234`
3. Kelola menu, lihat dashboard, export laporan
